# 134 sum 

有一个正整数，把它拆分成 1， 3, 4, 有限种排列的和， 求一个正整数n,
可以有多少种排列方式Dn。
例如：
n = 1:                                     => D1 = 1
n = 2: 2= 1+1                              => D2 = 1
n = 3: 3 = 3, 3 = 1+1+1                    => D3 = 2
n = 4: 4 = 4, 4 = 1+3, 4 = 3+1, 4 = 1+1+1, => D4 = 4
类似的，有D5=6, D6=9, 求Dn?
